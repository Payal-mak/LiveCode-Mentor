📖Explanation
This for loop starts at index i=0 and runs n times, adding each element of arr[] to sum.

It iterates through all array elements and accumulates their total — a classic running sum pattern.
⚠️Beginner-Friendly Error
Line 16: Uninitialized variable x

You declared int x but never assigned it a value before using it. Think of a variable like a box — you created the box, but put nothing in it. C++ doesn't automatically fill it with 0!
